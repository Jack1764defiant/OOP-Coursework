# WMS Interface Documentation

### Running the program

To run the program, you must have a jdk installed - you can find one here: https://www.oracle.com/uk/java/technologies/downloads/

Set up your IDE of choice for java - vscode needs an extension installed to support java.

Navigate to the main.java file and run it.

### Using the program

Upon running the program, you should see a menu like the below:
\
WMS Menu
1. Exit WMS interface.
2. Suppliers and Clients
3. Orders
4. Inventory
5. Financial
Enter option: 
\
\
Where it says enter option, enter a number between 1 and 5 to select the associated option from the menu - e.g. entering 1 will terminate the program, and 2 will open the suppliers and clients sub-menu.


### Low Stock Alerts

If in the main menu, you are seeing text like either of the following, you can hide this text in the following way:
 - Low on RAM - only 5 remain.
 - No items low on stock.
\
In the main menu, select option 4 (inventory)
\
In the inventory menu, select option 4 (Toggle Display Low Stock Alerts)
\
This will hide the low stock alerts. You can go through the same process to reenable it.
